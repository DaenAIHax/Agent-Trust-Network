{{- if not .Values.redis.internal }}

5. External Redis is expected. The DPoP JTI store, rate limiter and WebSocket
   pub/sub all depend on it — without Redis the broker degrades to in-memory
   mode and HA is effectively disabled across replicas.
{{- end }}

{{- if not .Values.vault.internal }}

6. External Vault is expected. Make sure the KMS credentials (token or
   Kubernetes auth role) are wired via the broker Secret or via
   broker.extraEnvFrom.
{{- end }}
